# GlobalVision Systems & Data Intern Challenge

## Overview
This repository contains the solution for the Systems & Data Intern Take Home challenge. The goal is to analyze Salesforce data (`accounts` and `support_cases`), transform it using Python/SQL, and generate business insights.

## Structure
- `data/raw/`: Contains the original JSON datasets provided.
- `notebooks/`: Jupyter notebooks for data exploration, SQL processing, and visualization.
- `src/`: (Optional) Source code for reusable scripts.
